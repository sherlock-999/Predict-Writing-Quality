# Linking Writing Process to Writing Quality

Kaggle competition: predict a holistic essay score (0.5 – 6.0) from keystroke log data.

## Project structure

```
writing_process/
├── data/                   # raw competition data (not tracked)
│   ├── train_logs.csv
│   ├── train_scores.csv
│   ├── test_logs.csv
│   └── sample_submission.csv
├── models/                 # saved fold models (not tracked, generated by train_lgbm.py)
├── eda_plots/              # output plots from eda.py
├── lgbm_plots/             # output plots from train_lgbm.py
├── eda.py                  # exploratory data analysis + feature correlation plots
├── train_lgbm.py           # 5-fold CV training + feature importance plots
├── predict.py              # inference on test set → submission.csv
├── requirements.txt
└── .gitignore
```

## Setup

```bash
conda create -n exp python=3.11
conda activate exp
pip install -r requirements.txt
```

## Usage

Run in order:

```bash
# 1. EDA — understand the data and feature correlations
python eda.py

# 2. Train — 5-fold CV, saves fold models to models/
python train_lgbm.py

# 3. Predict — generates submission.csv from test_logs.csv
python predict.py
```

## Features (19 total)

All features are computed per essay from the raw keystroke log.

| Category | Features |
|---|---|
| **Volume** | `final_word_count`, `total_events` |
| **Speed & Fluency** | `median_iki_ms`, `iki_std`, `typing_wpm` |
| **Pausing** | `pauses_over_2s`, `pauses_over_5s`, `mean_long_pause_ms` |
| **Revision** | `deletion_ratio`, `global_revision_count`, `local_revision_count` |
| **Punctuation** | `period_count`, `comma_count`, `semicolon_count`, `newline_count` |
| **Writing Momentum** | `wpm_early`, `wpm_middle`, `wpm_late`, `wpm_acceleration` |

> Note: all alphabetic characters in `text_change` are anonymised as `q`.
> Punctuation (`.`, `,`, `;`, `\n`) is preserved and used directly.

## Model

LightGBM regression with 5-fold stratified cross-validation.
Stratification is on score bins so every fold has a representative score distribution.
Final test predictions are the average of all 5 fold models.

Key hyperparameters: `num_leaves=31`, `min_child_samples=20`, `learning_rate=0.05`,
`feature_fraction=0.8`, `bagging_fraction=0.8`, early stopping after 50 rounds.
